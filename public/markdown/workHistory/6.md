#### 바루다 닥터

- 병원 운영과 환자 관리를 위한 바루다 닥터 서비스 개발
  - 예약 관리, 치료사 전문 차트, 직원 관리, 내원 현황 분석, 콘텐츠 발송 등 병원 운영 솔루션 화면 및 기능 개발
- React 기반 서비스 화면, 사용자 플로우, API 연동, 반응형 UI 구현
- 기존 Node.js 기반 백엔드 기능을 Spring Boot 기반 구조로 마이그레이션
- 링크커넥션 홈페이지 개발 및 운영 배포
  - GitHub Actions, AWS CloudFront 캐시 무효화, Slack Incoming Webhook
- 기술스택
  - React.js, Next.js, Redux, Recoil, React Query, Node.js, Spring Boot, GitHub Actions, AWS CloudFront
