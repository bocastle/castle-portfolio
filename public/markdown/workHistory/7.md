#### 소프트위즈

- Go 기반 CRM, Gateway, Client API, Client Portal, Office Portal 등 여러 서비스 모듈 개발 및 운영 대응
  - 결제, CRM, Gateway, 관리자 포털, 배포, 보안 검증, 운영 이슈를 함께 다루며 운영 서비스 흐름 개선
- 외부 결제 연동과 운영 안정화
  - 여러 서비스 모듈의 입출금, 콜백, 상태 처리 흐름을 연결하고 시간대·금액 정규화, 실패 재시도와 운영 로그를 보강
- 관리자 수동 실행 안정화
  - 권한과 운영 환경을 검증하고, 헬스체크, 중복 실행 방지, 일자별 실행 범위 제한을 적용
- CRM 고객 목록 조회 성능 최적화
  - EXPLAIN ANALYZE 기반으로 로그인 이력, 거래 이력, 계좌 정보 선계산 병목 분석
  - 전체 이력 데이터를 먼저 계산하던 구조를 페이지 대상 고객 선조회 후 부가 정보 조회 방식으로 개선
  - 기본 목록 조회의 초 단위 DB 실행 시간을 200ms대 수준으로 단축
  - 기존 검색/필터/정렬 동작은 기존 쿼리 흐름으로 유지해 운영 영향 범위 최소화
  - Go 테스트와 회귀 테스트를 추가해 변경 안정성 검증
- 보안 검증과 세션 이슈 분석
  - 접근 권한과 결제 상태 조회 검증 범위 및 테스트 케이스를 정리하고, 인증·세션 장애의 원인을 추적해 운영 대응
- 서비스 헬스체크와 배포 검증 자동화
  - 서비스와 백그라운드 작업의 헬스체크, Jenkins 배포 스크립트, 런타임 검증 흐름을 구성
- 운영 환경 연결 구조 검토
  - 외부 진입점과 기존 운영 환경을 연결하는 구조를 검토하고, 쿠키·CORS·장애 경계를 정리
- 기술스택
  - Go, Gin, Java, Spring Boot, MySQL, MariaDB, Redis, gRPC, Jenkins, Bash, Git
